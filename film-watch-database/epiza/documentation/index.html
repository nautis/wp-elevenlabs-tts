Installation

The first thing you need to do when you want to install a new WordPress plugin is to login to your site admin page. Once there, go to Plugins -> Add New. Click "Upload Plugin" button and upload the zip file from your local hard drive.

Settings

You can access all settings of the plugin from the Epiza->Settings page.

TMDB API

To activate TMDB integration you'll need to obtain an API key from your TMDB account. To obtain an API key, go to the API page in your TMDB account and fill out the application form. Once your application is approved, you'll be able to access your "API key" and "API Read Access Token" on this page.

To enable TMDB API in Epiza, copy&paste "API Read Access Token" to Epiza->Settings.

Importing Movies & TV Shows

If you have completed the TMDB integration correctly, the importer will be activated. Importing movies or TV shows is easy. You can use the search box to search for any movie or TV show in the TMDB database. Once you find what you're looking for, you can add it to your import list by clicking the "plus" icon on the posters.

After the import process is completed, you can easily see the imported movies or TV series on your site by clicking the links in the upper right corner. Movies and TV shows are listed similarly to your theme's blog page. Custom post types are a core feature of WordPress, and your theme should display them properly. If you are not satisfied with the look, we recommend trying other themes.

If you wish, you can modify the custom post type and taxonomy links by editing the url slugs in the Settings->Permalinks page;

Custom Post Types

There are 2 custom post type and 3 custom taxonomy in Epiza. Custom post types are very similar (movies and tv shows). They have their own "genre" taxonomy and share the "actor" taxonomy. All fields are automatically filled in after import. You can easily edit them manually later if you wish.
